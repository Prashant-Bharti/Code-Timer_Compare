# C++ Code Timer Web App

## 🚀 What it does:
A web app to paste and run C++ code, returning the output and time taken.

## 🧰 Tech Stack:
- Python (Flask)
- HTML/CSS/JS
- GCC compiler

## ⚙️ How to Run Locally

```bash
git clone https://github.com/your-username/code-timer.git
cd code-timer
pip install -r requirements.txt
python app.py
